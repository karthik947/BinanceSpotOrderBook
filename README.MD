# Binance Spot - VisualizeOrderbook

This project is created to demonstrate how to visualize live orderbook that's updated real-time using D3JS.

### How to run ?

Follow the instructions below.

- Install Node JS [link](https://nodejs.org/en/)
- Download and unzip this package.
- Go to the current folder on command promt and run below command

```bash
npm install
node app.js
```

- Open "http://127.0.0.1:4000/" in the browser.
